# 📊 Sales Performance Analysis Dashboard

## 📖 Introduction
This **Sales Performance Dashboard** was built from the ground up as an independent project. Rather than following a pre-made tutorial, I designed this solution from scratch to solve the problem of fragmented sales data. By taking raw, unorganized data, I performed the full end-to-end process: cleaning, modeling, and visualizing the data to create a custom business intelligence tool. 🚀

This project highlights my ability to take raw information and turn it into a compelling, interactive narrative that informs business decisions.

## 📂 Dashboard File
You can explore the final product here: **[Sales_Performance_Dashboard.xlsx](Sales_Performance_Dashboard.xlsx)**

## 🛠️ Excel Skills Used
The following features were utilized to bring this dashboard to life:
* 📉 **Pivot Tables & Charts:** Aggregating data and creating dynamic visuals.
* 🎛️ **Slicers:** Enabling real-time, interactive filtering for a better user experience.
* 🧮 **Formulas & Functions:** Dynamic calculations for KPIs and data manipulation.
* 🎨 **Custom Formatting:** Using K/M notation and consistent styling for professional readability.

## 📊 Dashboard Build

### 📈 KPI Overview
* **Total Sales:** $4.38M
* **Total Items Sold:** 15,616
* **Average Order Value:** $2,920
* **Return Rate:** 24.8%

### 🛠️ Visual Insights
* **Monthly Sales Trend (Line Chart):** Captures performance fluctuations throughout the year, helping identify high-performing months.
* **Regional Performance (Bar Chart):** Clearly highlights which regions (North, East, etc.) are leading in revenue.
* **Promotion Effectiveness (Bar Chart):** Analyzes the impact of different marketing campaigns (e.g., FREESHIP vs. WINTER15) to see which drives the most revenue.
* **Item Type Analysis:** A sorted bar chart prioritizing the top-performing products.

### 🧮 Data Logic & Transformation
* **Data Cleansing:** Performed extensive cleaning on raw data to ensure consistent naming conventions and accurate data types.
* **Model Relationships:** Used Pivot Table connections to ensure that when a user clicks a **Slicer**, every chart and KPI card on the dashboard updates simultaneously.
* **Calculated Fields:** Derived key metrics from raw transaction logs to provide summary statistics.

## 💡 Key Insights & Takeaways
* **Promotional Win:** The "FREESHIP" campaign is the clear leader in driving sales volume. 🚚
* **Operational Alert:** The 24.8% return rate is a significant metric that suggests a potential need for a deep dive into product quality or shipping reliability. ⚠️
* **Regional Dominance:** The North and East regions contribute most significantly to the company's total revenue. 🌍

## 🏁 Conclusion
This dashboard is a testament to the power of Excel for business analysis. By moving from raw data to an interactive interface, I’ve demonstrated how to identify trends and provide clear recommendations for decision-makers.
